# citystgeorge-hermes

A complete tourism website for **St. George, Utah** — the gateway to Greater Zion, the Mighty 5 national parks, and Southern Utah's red rock wonderland.

Built from scratch as a clean, responsive static HTML/CSS site.

## Pages

- **Home** — Welcome page with hero, stats, quick links to all sections
- **National Parks** — 6 parks: Zion (45 min), Bryce Canyon (2 hrs), Grand Canyon North Rim (2.5 hrs), Capitol Reef (3.5 hrs), Arches & Canyonlands (4 hrs)
- **State Parks & Lakes** — Snow Canyon, Sand Hollow, Quail Creek, Gunlock
- **Annual Events** — Huntsman World Senior Games, St. George Marathon, IRONMAN 70.3, Art Festival, Dixie Days, Trail Hero, and more
- **Things to Do** — Hiking, boating, OHV, golf, historic downtown, Tuacahn amphitheater, museums, family activities, rock climbing
- **Trip Planning** — Getting here, weather, lodging, dining, sample itineraries (3/5/7 day), packing checklist, FAQ
- **About** — Site info + history of St. George ("Utah's Dixie")

## Design

- Color palette: warm sandstone + deep Utah sky + cream
- Full-width hero images from Unsplash
- Responsive grid layouts for cards
- Mobile-friendly hamburger nav
- No frameworks — pure HTML, CSS, and vanilla JS

## Deployment

Ready to deploy as a static site. Just upload the entire repo to any static host:

- **GitHub Pages** — push to `main` and enable Pages
- **Netlify** — drag the folder or connect the repo
- **Vercel** — connect the repo
- **Any web server** — upload the files as-is

No build step required. No dependencies.
